-> This project includes two mods: one implementing protein powder, and the other implementing santa. The link to the protein powder mod is here: https://github.com/dmooshm/ProteinPowder

-> Reflection
a) I found that C# was, for the most part, easy to pick up given my java experience. Syntactically, the languages are very similar, so there was a minimal learning curve. I used Microsoft Learn to initially orient myself with the language, but whenever I forogt anything, I would just google the java equivalent.
b) I only had to implement the basic stardew classes to make my mod run in the game.
c) I struggled a lot with file organization. My file explorer and rider would not sync, which resulted in the game not being able to locate certain files. I spent most of my time troubleshooting this; essentially, most of my work was actually getting a mod to run, rather than actually getting to work on the mod.
Unfortunately, this meant that I couldn't spend as much time as I would have wanted to on my project; particularly, in that I could only really makes a 'barebones' mod that couldn't implement any of the work (like pixelart that I'd done previously).
